1. Flex
> 父级元素设置
```js
display: flex;
justify-content: center;
align-items: center;
```

2. 定位 + transform
> 子元素
```
position: relative;
top: 50%;
left: 50%;
transform: translate(-50%, -50%);
```

3. 定位 + margin
> 子元素
```
position: relative;
left: 50%;
top: 50%;
margin-left: -50px;
margin-top: -50px;
```
