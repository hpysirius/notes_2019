## 1.不使用临时变量交换两个变量的值（异或，加减，乘除）

### 交换两个变量，不使用临时变量
#### 异或
```
a = a^b;
b = a^b;
a = a^b;
```
规则：
- 相同取0，相异取1

* 例子：
```
var a = 2;   010
var b = 3;   011
a = a ^ b    001
b = a ^ b    010    = 2
a = a ^ b    011    = 3
```

#### 加减
```
a = a + b;
b = a - b;
a = a - b;
```
#### 乘除
```
a = a * b;
b = a / b;
a = a / b;
```
