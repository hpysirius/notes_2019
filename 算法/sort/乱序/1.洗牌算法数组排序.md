### sort 结合 Math.random
```
function shuffle(arr){
    return arr.sort(() => Math.random() - 0.5);
}
```

### Fisher–Yates
* 将数组从后向前遍历，然后将当前元素与随机位置的元素进行交换
```
function shuffle(arr) {
    let m = arr.length;
    while(m){
        const index = Math.floor(Math.random() * m--);
        const temp = arr[m];
        arr[m] = arr[index];
        arr[index] = temp;
    }
    return arr;
}
```
